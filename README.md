# 個人 AI 發文器（骨架）

> 從 GitHub 下載這個骨架，給 Cowork 你的個人 DNA（個人說明書 / 職務說明書 / 粉專備份），AI 自動幫你客製化 + 部署成**屬於你自己的 FB 發文工具**。

---

## 適合誰

不是只給電商主用。任何需要常發 FB 長文的人都適用：

| 你是誰 | 你的工具會被客製成 |
|---|---|
| 電商主 / 團媽團爸 | 團購文產生器 |
| 知識型 KOL / 教練 / 諮詢師 | 課程招生文 / 個案故事產生器 |
| 部落客 / 媽媽 / 寵物店 | 生活分享文產生器 |
| 公司小編 | 品牌貼文產生器 |
| 任何想用 AI 發 FB 長文的人 | 你的版本 |

**同一個骨架，每個人客製化結果不一樣**。

---

## 跟另一個業配版的差別

- 業配版 ([tg-writer-template](https://github.com/Summerhuang585/tg-writer-template))：寫死 10 段式團購文格式，適合「我就是要做業配/帶貨用」的人
- 這個骨架（個人版）：**三套文章結構模組（電商 / 知識 / 生活）任選或自由**，加上 Cowork 用你 DNA 客製化

如果你只做業配/團購，用業配版直接部署。如果你想要「屬於你自己」的工具，用這個骨架。

## 怎麼開始

最快的路：用 **Claude 桌面 Cowork 模式** + 我們準備好的 prompt，30 分鐘搞定。

👉 **完整步驟看 [用Cowork客製化部署.md](用Cowork客製化部署.md)**

簡述流程：
1. 從 GitHub 下載這個 repo 的 ZIP，解壓縮
2. 開 Claude 桌面版 → 切到 Cowork 模式 → 把資料夾連到 Cowork
3. 準備好你的三份個人 DNA 素材（看下方）
4. 複製 [用Cowork客製化部署.md](用Cowork客製化部署.md) 裡的 prompt 給 Cowork
5. Cowork 會問你問題、改 code、用 Netlify MCP 部署，最後給你網址

## 你需要準備的三份個人 DNA 素材

這是 Cowork 客製化工具的「燃料」，越完整 Cowork 寫出來的工具越像你。

### 1. 個人說明書（5-15 分鐘寫完）

簡單回答：
- 我是誰（名字、頭銜、年紀）
- 我的品牌叫什麼
- 我的客群是誰（年齡、性別、生活情境）
- 我的核心優勢（為什麼選我而不選別人）

範本看 [DNA萃取範本.md](DNA萃取範本.md)。

### 2. 職務說明書（10-20 分鐘寫完）

回答：
- 我賣什麼（產品 / 服務 / 課程 / 知識）
- 我的客人在哪買（FB / IG / LINE 群 / 自己網站）
- 我的運作模式（結團週期、結帳流程、客服 ID 等）
- 我每篇 FB 文最常包含的元素

範本一樣在 [DNA萃取範本.md](DNA萃取範本.md)。

### 3. 粉專/IG/Threads 備份（5 分鐘導出）

到你最常發文的平台導出過去 6-12 個月的貼文：
- FB：設定 → 你的 Facebook 資訊 → 下載個人檔案 → 只勾「貼文」→ 下載 HTML 或 JSON
- IG：設定 → 隱私和安全 → 申請下載資訊
- Threads：類似流程

或者最簡單：複製 20-50 篇你寫過最滿意的文章到一個文字檔，丟給 Cowork 用。

## 部署需要

| 項目 | 需求 |
|---|---|
| Claude 桌面版 | 含 Cowork 模式 |
| Netlify 帳號 | Pro（$19/月，要 Background Function 跟 Blobs） |
| Anthropic API Key | 自備（每篇文 NT$ 0.5-20，依模型而定） |

---

## 技術棧

- 前端：純 HTML / CSS / JS
- 部署：Netlify Pro + Background Functions（15 分鐘 timeout）
- 儲存：Netlify Blobs（雲端 KV）
- AI：Anthropic Claude API（Haiku 4.5 / Sonnet 4.5 / Opus 4.6）
