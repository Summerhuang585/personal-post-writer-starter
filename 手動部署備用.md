# 手動部署（不用 Cowork 的備用方案）

如果你不會用 Cowork，或想全程自己掌控，照這份做。

預計時間：30-45 分鐘。

---

## Step 1 · 下載這個 repo

到 https://github.com/Summerhuang585/personal-post-writer-starter

點 **`<> Code`** → **Download ZIP** → 解壓縮。

---

## Step 2 · 自己萃取 DNA（不用 Cowork 幫你）

照 [DNA萃取範本.md](DNA萃取範本.md) 寫好三份檔案。

接著開 Claude 網頁版（不是 Cowork，是普通的 chat 模式），把三份 DNA 素材貼進去，問 Claude：

```
請從這三份個人 DNA 素材幫我萃取以下資料,輸出成可以直接貼到環境變數的格式:

1. POST_TYPE(從以下選一個):電商團購 / 知識教學 / 生活分享 / 自由
2. BRAND_NAME(我的品牌名)
3. FIRST_PERSON(我的第一人稱,例:姐 / 我 / 老闆娘)
4. PERSONAL_VOICE(我的口頭禪 top 10,用逗號分隔)
5. PERSONAL_ROLES(我常提到的角色,用逗號分隔)
6. LINE_ID(我的 LINE 客服 ID)

[貼入三份 DNA 素材]
```

Claude 會回給你 6 個值。

---

## Step 3 · 申請 Anthropic API Key

- 到 https://console.anthropic.com 開帳號
- API Keys → Create Key → 複製 `sk-ant-xxx...`
- Plans & Billing → 儲值 5-10 美金

---

## Step 4 · 開 Netlify Pro 帳號

- https://app.netlify.com → 註冊 → Upgrade Pro $19/月

---

## Step 5 · 拖檔到 Netlify

- Sites → Add new site → Deploy manually
- 把資料夾拖進去（**只拖 code 檔，DNA 素材不要拖**）
- 等 30 秒部署完
- Change site name → 改成你要的網址

---

## Step 6 · 創 Netlify PAT

- https://app.netlify.com/user/applications#personal-access-tokens
- New access token → 取名 → No expiry → Generate → 複製 token

---

## Step 7 · 設環境變數

到 Site → Site configuration → Environment variables，加以下 10 個：

| Key | Value | 來源 |
|---|---|---|
| ACCESS_PASSWORD | 自己想一組密碼 | 你 |
| OPUS_PASSWORD | 自己想一組 Opus 密碼 | 你 |
| ANTHROPIC_API_KEY | sk-ant-xxx... | Step 3 |
| NETLIFY_BLOBS_TOKEN | nfp_xxx... | Step 6 |
| POST_TYPE | 電商團購 / 知識教學 / 生活分享 / 自由 | Step 2 Claude 給的 |
| BRAND_NAME | 你的品牌名 | Step 2 |
| LINE_ID | @xxx | Step 2 |
| FIRST_PERSON | 姐 / 我 / etc. | Step 2 |
| PERSONAL_VOICE | 真的,超,絕對,... | Step 2（逗號分隔） |
| PERSONAL_ROLES | 我自己,我家小孩,... | Step 2（逗號分隔） |

每個變數 Scopes 全部勾「All scopes」。

---

## Step 8 · Redeploy 讓變數生效

Deploys → Trigger deploy → **Clear cache and deploy site**

等 30 秒。

---

## Step 9 · 開來用

打開 `https://你的站名.netlify.app` → 輸入 ACCESS_PASSWORD → 填表單產文。

---

## 想再深度客製怎麼辦

如果你的領域很特殊（例如做寵物殯葬服務、做老人照顧用品），AI 寫的可能還不夠像你。

打開 `netlify/functions/generate-background.js`，找到 `buildSystemPrompt()` 函數，自己改：

- 文章結構 A / B / C：留你領域對應那段，刪其他
- 禁區：改成你領域的禁區
- 加入你的獨家寫作公式

改完整個資料夾重新拖到 Netlify 覆蓋上版。
