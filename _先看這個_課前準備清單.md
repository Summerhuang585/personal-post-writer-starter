# 課前準備清單（請先看這份）

> 上課前先把這些做完，課堂才能跟得上。預計準備時間 **30-60 分鐘**。

---

## 你要準備的東西

### A · 帳號 / API（共 3 個）

| 帳號 | 怎麼開 | 預計花費 |
|---|---|---|
| **Anthropic API key** | 看下方 [Step 1](#step-1--申請-anthropic-api-key)（5 分鐘） | 儲 5-10 美金 |
| **Netlify Pro 帳號** | 看下方 [Step 2](#step-2--開-netlify-pro-帳號)（5 分鐘） | $19 美金 / 月 |
| **Claude 桌面版**（含 Cowork） | https://claude.ai/download（3 分鐘） | 免費 |

### B · 個人 DNA 素材（最重要，3 份檔案）

| 檔案 | 內容 | 寫多久 | 範本 |
|---|---|---|---|
| `個人說明書.md` | 我是誰、我的品牌、我的客群、我的優勢 | 10 分鐘 | [DNA萃取範本.md](DNA萃取範本.md) |
| `職務說明書.md` | 我賣什麼、運作模式、禁區 | 15 分鐘 | [DNA萃取範本.md](DNA萃取範本.md) |
| `粉專備份.txt` | 你過去最滿意的 20-50 篇 FB / IG / Threads 貼文（完整內文，每篇用 === 分隔） | 20-30 分鐘 | 看下方範例 |

⚠️ **粉專備份不到 20 篇，AI 萃取出來的 DNA 不會準。** 越多越好，建議 30-50 篇。

## Step 1 · 申請 Anthropic API Key

這是讓你的工具能跑 AI 的「鑰匙」。

1. 開瀏覽器 → https://console.anthropic.com
2. **Sign up** → 用 Google 帳號登入最快
3. 登入後，右上角頭像 → **API keys**
4. 點 **Create Key**
5. 取個名字（隨便寫，例：`mywriter-key`）→ **Generate**
6. **複製跳出來的 `sk-ant-xxxxx...`**（只會顯示一次！沒存到要重新產一把）
7. 存到記事本

### 還要儲值（不儲值會跑不動）

8. 左邊選單 **Plans & Billing** → 綁信用卡
9. 儲值 **5-10 美金** 先試（用多少扣多少）

10 美金大概可以跑 100-500 篇 Sonnet 4.5 文章，足夠你測試一兩個月。

---

## Step 2 · 開 Netlify Pro 帳號

工具要放在網路上，這就是「網路上的房子」。

1. 開 https://app.netlify.com
2. **Sign up** → 用 Google 登入最快
3. 登入後右上頭像 → **Team settings** → **Billing** → **Upgrade to Pro**
4. 信用卡填一填，月費 $19 美金

⚠️ **必須 Pro，免費版功能不夠**，AI 跑到一半會被切斷。

---

## Step 3 · 下載工具骨架

到 GitHub 下載：

👉 https://github.com/Summerhuang585/personal-post-writer-starter

點右上綠色按鈕 **`<> Code`** → **Download ZIP** → 解壓縮到電腦。

得到資料夾：`personal-post-writer-starter-main`

## Step 4 · 寫你的三份 DNA 素材

到剛解壓縮的資料夾，新建三個檔案：

### 4-1 · 個人說明書.md

用記事本或 Word 開，複製 [DNA萃取範本.md](DNA萃取範本.md) 「範本 1」段落，把【】內容填好。

### 4-2 · 職務說明書.md

同樣複製 [DNA萃取範本.md](DNA萃取範本.md) 「範本 2」段落填好。

### 4-3 · 粉專備份.txt

最簡單做法：

1. 開個記事本
2. 到你 FB 粉專 / IG / Threads
3. **複製過去 6-12 個月寫得最滿意的 20-50 篇貼文**完整內文
4. 一篇一篇貼進記事本，每篇之間用 `===` 三個等號分隔
5. 存成 `粉專備份.txt`

格式範例：

```
===
這款護手霜真的拿出來說有面子!姐用一週手就嫩回來!
[完整內文]
===
為什麼這條會成為妳化妝包的新寵?
[完整內文]
===
[第三篇...]
```

**沒有 20 篇？** 補到 20 篇再來，不然 AI 萃取會失準。

## 上課當天會做什麼

1. 老師教你開 Cowork、連資料夾、連 Netlify MCP
2. 老師給你一段 Mega Prompt 複製貼到 Cowork
3. Cowork 跟你互動 30-45 分鐘：
   - 讀你的 3 份 DNA 素材
   - 萃取你的個人 DNA 摘要
   - 改寫工具的 SYSTEM_PROMPT
   - 用 Netlify MCP 自動部署
   - 設好所有環境變數
4. 你拿到屬於你的網址 → 試一篇文章 → 收工

整堂課大約 1.5-2 小時。

## 沒做完作業會怎樣

| 沒做的東西 | 課堂後果 |
|---|---|
| 沒升 Netlify Pro | Cowork 跑到一半失敗 |

| 粉專備份不到 20 篇 | AI 萃取的 DNA 不準，寫出來的文不像你 |
| 沒裝 Claude 桌面版 | 完全沒辦法用 Cowork |

**請提前做完，不要當天才弄。**
